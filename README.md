# PowerBI-Sales-Dashboard
Advanced Sales Dashboard using Power BI for Microsoft Elevate Internship

## Project Overview
This project is an interactive Power BI dashboard designed to analyze sales data and generate business insights.

## Features
- Total Sales, Profit and Quantity KPIs
- Year-wise Sales Trend Analysis
- Customer Segment Analysis
- Payment Mode Distribution
- Regional Sales Map
- 15-Day Sales Forecast

## Tools Used
- Microsoft Power BI
- Power Query
- DAX

## Dataset
Superstore Sales Dataset
